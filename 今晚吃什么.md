# 今晚吃什么

产品定位：解决用户「今晚该吃什么」的选择困难——用户用一句话输入早/午餐，产品给出**决断式**的晚餐推荐（一个明确答案 + 精炼理由）。

**目标**：上线一个网页版 MVP，跑通「输入早/午餐 → 决断式晚餐答案 + 营养理由」核心闭环。
**1.0 范围（已锁定）**：LLM 为主 / 网页优先 / 核心闭环 only。放纵博弈、二次传播趣味、拍照识别、小程序端列为 fast-follow。
**产出**：方案 [[1.0方案]] + 未决项 [[待澄清问题]] + 技术选型 [[后端技术选型]]；M1–M4 核心闭环已落为可运行代码，见 [[app/app|app]]（本地端到端可跑，mock/真实模型一键切换）。
